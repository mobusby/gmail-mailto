About this project:

gmail-mailto.py is the 'mailto:' link handler
Gmail-mailto.desktop opens a normal Gmail session in chromium-broswers app-mode
Gmail_logo.png is a gmail icon file, borrowed from Gmails google+ page


To use:

Place Gmail-mailto.desktop in `$HOME/.local/share/applications/`
Place Gmail_logo.png in `$HOME/.local/share/icons/`
Place gmail-mailto.py in your `bin` directory (recommend $HOME/.local/bin)
